Данный репозиторий предназначен для ознакомительной демонстрациии проектов, которые были мной реализованы.
Для недопущения разглашения коммерческой тайны все данные были предварительно изменены.

Файл "Реализация нп по дням" предназначен для наглядного сравнения реализации (в данном случае нефтепродуктов) в любых периодах между собой с разбивкой по любым группам или категориям. Желтые ячейки могут быть изменены. Использованы функции СУММ, СУММЕСЛИМН, ГПР, ДВССЫЛ, а так же использовано условное форматирование.

Файл "Расчет премирования сотрудников" предназначен для итогового расчета премии сотрудникам по различным показателям. Использованы функции ЕСЛИ, СУММЕСЛИ, а так же использовано условное форматирование.

Файл "Сводная по розничной реализации НП" представляет собой сводную таблицу по данным за несколько периодов. В сводной таблице для наиболее удобного и быстрого построения реализовано использование срезов.
